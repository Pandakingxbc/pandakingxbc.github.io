## PaperTrack-Agent: Multi-Agent Research Assistant

**Duration:** Oct 2025 - Dec 2025
**Tech Stack:** Python + LangGraph + LLM + Redis + MySQL + Streamlit

### Overview

A multi-agent system for automated paper tracking and research assistance, featuring 10 specialized agents for paper discovery, analysis, trend detection, and research gap identification.

### System Architecture

- **Multi-Agent Framework**: Built on LangGraph state machine for workflow orchestration
- **Paper Processing Pipeline**: Automated paper retrieval, parsing, and analysis
- **Semantic Search**: ChromaDB vector database for similarity matching

### Key Features

1. Automated paper discovery and retrieval
2. LLM-powered paper structure parsing and method comparison
3. Research trend analysis and gap identification
4. Modular design supporting easy agent extension
