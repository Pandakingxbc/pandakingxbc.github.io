## Multimodal Vision-Language Navigation System

**Duration:** Sep 2025 - Jan 2026
**Type:** Course Project

### Overview

A Transformer-based Vision-Language Navigation system that enables robots to navigate in 3D environments based on natural language instructions.

### Technical Architecture

- **Text Encoding**: BERT for natural language instruction understanding
- **Visual Encoding**: CLIP for visual feature extraction
- **Depth Information**: Fused with visual features for 3D understanding
- **Topological Planning**: NetworkX for graph-based shortest path navigation

### Implementation Details

1. Multi-modal Transformer architecture for cross-modal alignment
2. Topological node-based navigation prediction
3. Dynamic environment graph construction
4. Validated on R2R-CE dataset

### Learning Outcomes

- Deep understanding of Transformer applications in multi-modal tasks
- Experience with topological planning for robot navigation
- Familiarity with state-of-the-art embodied AI and VLN techniques
