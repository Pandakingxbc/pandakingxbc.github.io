## Air-Ground Heterogeneous Indoor Navigation and Detection

**Duration:** Oct 2024 - Present
**Type:** National Key R&D Program

### Overview

A semantic-driven autonomous UAV exploration system for post-earthquake damaged buildings where human operators cannot safely enter.

### System Pipeline

Complete embodied navigation closed-loop:
1. **Environment Perception** → Multi-modal sensing
2. **Semantic Mapping** → Semantic Occupancy Map construction
3. **LLM Planning** → High-level task planning
4. **MPC Control** → Low-level motion control

### My Contributions

#### 1. Semantic Perception and Mapping
- Fused visual and depth information for environment perception
- Constructed Semantic Occupancy Map with semantic labels in ESDF map
- Provided interpretable spatial semantic information for navigation decisions

#### 2. LLM Agent Decision
- Built Embodied Agent driven by Large Language Model
- Integrated semantic map with multi-modal perception for high-level task planning
- Implemented exploration strategy driven by semantic association and information gain
